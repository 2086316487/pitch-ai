#### 🐛 修复的问题

1. **ReactMarkdown插件懒加载错误**
   - 问题：尝试懒加载remarkGfm插件导致"Expected usable value but received an empty preset"错误
   - 影响：用户无法查看生成的完整商业计划书
   - 解决：回退到常规导入，react-markdown插件系统不支持动态加载
   - 文件：app/plan/page.tsx

2. **财务模型API导入错误**
   - 问题：导入了不存在的callOpenAI函数，导致"Export doesn't exist"错误
   - 影响：用户无法生成财务模型
   - 解决：改用直接导入openai客户端，实现本地withRetry函数
   - 优化：简化prompt结构，添加详细调试日志
   - 文件：app/api/generate-financial-model/route.ts

3. **Next.js配置警告**
   - 问题：使用了已废弃的swcMinify和compress选项
   - 解决：移除这些选项（Next.js 15中默认开启）
   - 文件：next.config.ts

   - 解决：实现自动重试机制（指数退避策略）+ 增加超时时间

   - 问题：AI 返回 `reasoning_content` 而非 `content` 字段
   - 解决：兼容处理两种字段 + 增加 max_tokens 到 4000

   - 问题：思考内容过长导致实际输出被截断
   - 解决：调整 prompt 策略 + 使用 system message
